# ChatPlace 🚀

A modern, beautiful website for Instagram and Telegram integration built with HTML, CSS, and JavaScript. Created by **NetGlobal Team**.

## Features ✨

- **Light Blue & Transparent Design** - Modern UI with a light blue color scheme and glassmorphism effects
- **Dark Mode** - Full dark mode support for comfortable viewing at night
- **Responsive Design** - Works perfectly on all devices (desktop, tablet, mobile)
- **Smooth Animations** - Beautiful fade-in and scroll animations throughout
- **Social Media Integration** - Quick links to Instagram and Telegram
- **Contact Form** - Interactive contact form with validation
- **Keyboard Shortcuts** - Alt+D for dark mode toggle, Alt+H for home
- **Smooth Scrolling** - Navigate smoothly between sections

## Project Structure 📁

```
chatplace/
├── index.html       # Main HTML file with page structure
├── styles.css       # CSS styling with dark mode support
├── script.js        # JavaScript for interactivity
└── README.md        # This file
```

## Getting Started 🚀

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or dependencies required!

### Installation

1. Clone the repository:
```bash
git clone https://github.com/somystra/chatplace.git
cd chatplace
```

2. Open the website:
```bash
# Simply open index.html in your browser
# Or use a local server:
python -m http.server 8000
# Then visit: http://localhost:8000
```

## Features in Detail 🎨

### Light Blue & Transparent Theme
- Primary color: `#87CEEB` (Sky Blue)
- Uses CSS glassmorphism with `backdrop-filter: blur()`
- Semi-transparent backgrounds throughout

### Dark Mode
- Toggle with the moon/sun button in the header
- Automatic theme persistence using localStorage
- Smooth transitions between themes
- Keyboard shortcut: `Alt + D`

### Responsive Sections

1. **Header** - Logo with subtitle and dark mode toggle
2. **Navigation** - Quick access to all sections
3. **Hero Section** - Welcome message and call-to-action
4. **Features** - Grid layout showcasing 4 main features
5. **Social Links** - Instagram and Telegram connections
6. **Contact Form** - Interactive form with validation
7. **Footer** - Copyright information

### Interactive Elements

- **Hover Effects** - All buttons and cards have smooth hover animations
- **Form Validation** - Email and required field validation
- **Scroll Animations** - Cards fade in when scrolled into view
- **Active Navigation** - Shows which section you're currently viewing
- **Ripple Effect** - Material Design ripple effect on button clicks

## Customization 🎯

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #87CEEB;
    --primary-dark: #4A90E2;
    /* ... more variables */
}
```

### Update Social Links
Edit the href attributes in social buttons in `index.html`:
```html
<a href="YOUR_INSTAGRAM_URL" class="social-btn instagram">
```

### Modify Content
All text content can be easily edited in `index.html`.
